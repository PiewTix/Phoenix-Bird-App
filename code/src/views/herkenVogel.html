Vogel Herkennen

Hoe werkt het?


Per onderdeel van de vogel kan je een aantal kenmerken aanduiden. Indien je twijfelt, kan je deze laten staan op onbekend.
Bij het onderdeel Resultaten zullen de namen van de vogels gegeven worden die overeen komen met jouw ingegeven kenmerken.
Je kan op deze namen drukken voor meer informatie te krijgen over de vogel.
Ook is er reset knop voorzien om alle kenmerken terug op onbekend te plaatsen indien je dit wenst.
